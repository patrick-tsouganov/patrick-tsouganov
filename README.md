## 📌 Featured Projects

### 🎓 Automated Extraction of Locomotion Loop Segments from Raw Motion Capture Data
> *MSc Thesis — In Progress*

Investigating whether machine learning can automate the extraction of locomotion loop boundaries from raw, unsegmented motion capture sequences — a process that currently relies on manual keyframe identification. Comparing four sequence models (LSTM, TCN, Transformer encoder, and Mamba) against a signal-based heuristic baseline, using seam quality as the primary evaluation metric. Data sourced from the CMU Graphics Lab Motion Capture Database and AMASS dataset.

`Python` `PyTorch` `LSTM` `Transformer` `Mamba` `Motion Capture` `Deep Learning`

---

### 🔬 Predicting Photogrammetry Reconstruction Quality from Raw Image Statistics
> *ML Research Project*

Investigated whether the quality of a 3D photogrammetric reconstruction can be predicted from simple statistics derived from raw input images, before any reconstruction pipeline is run. Built an end-to-end pipeline covering MVS reconstruction across ETH3D scenes, extraction of 34 low-level image features per scene, and training and evaluation of seven models of escalating complexity. Quality measured as F1 score at 2cm tolerance against ground truth laser scans.

`Python` `Scikit-learn` `Computer Vision` `Machine Learning` `Photogrammetry` `ETH3D`
